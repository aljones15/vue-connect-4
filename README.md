# Vue.JS Connect 4

A simple implementation of Connect 4
Features:
- [X] player versus player
- [X] player vs ai

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Lints and fixes files
```
yarn run lint
```
